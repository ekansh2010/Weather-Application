# 🌦️ SkyCast — Modern Weather Experience

<div align="center">

### Real-Time Weather Forecasting with Dynamic Visual Effects

SkyCast is a modern weather application that delivers live weather updates through an immersive glassmorphism interface, animated weather visuals, and responsive UI interactions powered by the OpenWeatherMap API.

# ⚙️ Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/weather-app.git
```

## Navigate to Project

```bash
cd weather-app
```

## Add API Key

Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

inside `script.js` with your OpenWeatherMap API key.
